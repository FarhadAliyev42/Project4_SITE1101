What We Did

Built basic logic gates (NOT, AND, OR, NAND, XOR) using transistors and electronic components on a solderless breadboard. Demonstrated how fundamental digital logic operates in practice.

  • Constructed NOT, AND, OR gates using NPN transistors (PN2222)
  • Built NAND gate by collaborating with Team 9
  • Built XOR gate by collaborating with Teams 7, 8, 9
  • Recorded and edited video demonstrating the construction process

What We Did

Built an autonomous robot using LEGO Mindstorms EV3 that draws artwork on paper.

  • Assembled the Base Educator model with gyro sensor for precise navigation
  • Engineered a custom marker holder for drawing
  • Planned artwork and programmed robot's pathing using MakeCode JavaScript
  • Managed motor speeds and sensor feedback for accurate drawing on A1 surface
  • Documented construction and testing process in a narrated video
