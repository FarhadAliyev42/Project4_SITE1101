## About the Project

This is a fully responsive, single-page personal portfolio website developed as part of the SITE 1101 course at ADA University. The site showcases my academic background, technical skills, course projects, homework assignments, and includes interactive JavaScript games as a bonus feature.

The entire website was developed using **Claude AI** (via the VSCode extension) as an AI coding assistant. All AI conversations during development have been documented and submitted as part of the project requirements.

---

## Features

- **Single-page layout** with smooth scroll navigation
- **Responsive design** that adapts to all screen sizes
- **Interactive home page** with typewriter effect, animated counters, and scroll-reveal cards
- **Project modals** with team details, descriptions, photo galleries, and YouTube video links
- **Lightbox photo gallery** for each project with keyboard navigation
- **Homework cards** with course descriptions, report downloads, and Codecademy links
- **3 playable JavaScript games** (Snake, 2048, Memory Match)
- **Modular and clean code structure**

## Development Process

This website was developed step by step using **Claude AI** in VSCode as the primary coding assistant. Key development phases:

1. **Environment setup** — VSCode, Live Server extension, Git repository, folder structure
2. **Home page** — Hero section, navigation bar, footer, profile photo, animated elements
3. **About page** — Education cards, school journey timeline, skills, achievements, social links
4. **Projects page** — Project cards, team modals, YouTube links, photo galleries with lightbox
5. **Homework section** — Course cards with report downloads and Codecademy links
6. **UI/UX polish** — Modal animations, card alignment, responsive fixes, hover effects
7. **Games section** — Snake, 2048, and Memory Match games added as bonus
8. **GitHub Pages** — Repository pushed and site deployed

All AI conversations during development are documented in `ai-chat.md` (submitted separately).
